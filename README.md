# API Node Rest

## Clone Repository from GitHub

```bash
> git clone git@github.com:fedearbe/api-node-dev.git
```

## Create file .env and next lines

```bash
SERVER_HOST="127.0.0.1"
SERVER_PORT=3000

DB_CONNECTION_STRING=""
DATABASE_PROTOCOL=""
DATABASE_URL=""
DATABASE_USERNAME=""
DATABASE_PASSWORD=""
```

## Install application

```bash
> npm install
```

## Run Server

```bash
> npm run dev
```
